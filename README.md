# F1-AI-Data-Visualizer
This is a fun project that's part of my portfolio, it combines one of my interests, Motorsport, with my dream career, SWE. The idea is simple, take data, run it through a summarization and use a model like claude for visualization and explanation. 



# The Backend 
The backend is built on FastAPI (https://fastapi.tiangolo.com/), a fast, simple to work with framework that I find perfect for this project, alongside Uvicorn, which is amazingly optimized for FastAPI.

For this project, OpenF1 is used for pulling race data that can be then processed by Gemnini or Claude for data visualization in the frontend.


# The Frontend
The frontend is a simple, react web application that uses oxlint for its speed and simplicity, Tailwind CSS for it's beautiful inline customization, and regular CSS for constant styles.
